# Algorithms Analysis Lab Project

A Java-based project analyzing algorithm design and complexity, implemented as two Maven modules: a core logic module and a GUI-based version.

## Overview
This project was developed for an "Analysis of Algorithms" course, focusing on algorithm design, implementation, and complexity analysis. It includes both a console-based implementation and a GUI version.

## Tech Stack
- Java
- Maven

## Project Structure
```
FinalAlgorithmProject/      # Core algorithm implementation
FinalAlgorithmProjectGUI/   # GUI version
Final Algorithm Lab Project.pdf/.docx  # Written report
```

## How to Run
```bash
cd FinalAlgorithmProject
mvn compile
mvn exec:java
```
